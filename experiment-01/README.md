# Experiment 01: Basic Statistical Measures

## Topic
Core statistical computations using small sample arrays.

## Objective
Implement and understand foundational statistical formulas in Python.

## Dataset
No external dataset is used. The code uses hardcoded sample arrays.

## Code Flow
1. Compute expected value from values and frequencies.
2. Compute mean of a numeric array.
3. Compute standard deviation.
4. Compute variance.
5. Compute covariance between two arrays.
6. Compute covariance matrix.

## Libraries Used
- NumPy

## Output
- Printed numeric outputs for each statistical measure.
- Covariance matrix displayed in array form.
