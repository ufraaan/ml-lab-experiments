# ML Lab Experiments

A collection of machine learning experiments implemented in Jupyter notebooks, covering various datasets and problem domains.

## Requirements

 - Python 3.14.2 or above
- Jupyter Notebook / JupyterLab
- See individual notebooks for experiment-specific dependencies

## Usage
```bash
git clone https://github.com/moroii69/ml-lab-experiments.git
cd ml-lab-experiments
jupyter notebook
```
