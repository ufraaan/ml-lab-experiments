# Experiment 06: Logistic Regression Classification

## Topic
Binary classification for purchase prediction.

## Objective
Build and evaluate a logistic regression classifier after encoding and scaling features.

## Dataset
- `data/user_data.csv`

## Code Flow
1. Load user dataset.
2. Drop non-informative ID column.
3. Encode categorical feature(s) such as gender.
4. Split into train and test sets.
5. Scale feature values.
6. Train logistic regression model.
7. Predict test labels.
8. Evaluate using accuracy, confusion matrix, and classification report.
9. Display sample predictions.

## Libraries Used
- Pandas
- NumPy
- scikit-learn

## Output
- Classification performance metrics.
- Confusion matrix and class-wise report.
- Prediction examples.
