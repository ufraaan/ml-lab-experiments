# Experiment 03: Exploratory Data Analysis and Visualization

## Topic
Visual analysis of a book dataset using multiple plot types.

## Objective
Perform EDA and generate meaningful visualizations to understand pricing, ratings, and trends.

## Dataset
- `data/book_data.csv`

## Code Flow
1. Load dataset and clean column names.
2. Preview data and inspect data types.
3. Compute summary statistics.
4. Create bar chart for price by author.
5. Create scatter plot for pages vs price.
6. Create box plot for price distribution.
7. Create histogram for rating distribution.
8. Create line graph for yearly average price trend.

## Libraries Used
- Pandas
- Matplotlib

## Output
- EDA tables and summary.
- Multiple charts saved and displayed in notebook.
